About This Project

The Fabric Image Classifier is an AI-powered web application that uses deep learning to automatically identify different types of fabrics from images. Using a Convolutional Neural Network (CNN) with transfer learning, the system can accurately classify fabrics into various categories.

Model Architecture

The classifier uses MobileNetV2 as a base model, pre-trained on ImageNet. This is enhanced with custom dense layers and dropout for fabric-specific classification. The model is trained with data augmentation to improve generalization.

Getting Started

To use this application:

1. Train the Model

Run python train_model.py to train the classification model on your fabric dataset

2. Start the Web App

Run python app.py to start the Flask web server

3. Upload & Analyze

Open your browser, upload a fabric image, and get instant predictions
